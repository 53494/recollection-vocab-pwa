# 开发需求文档 - Recollection

## 项目概述

**Recollection** 是一款面向英语专业学生的深度学习型词汇记忆 PWA 应用。区别于传统背单词软件（不背单词、墨墨、百词斩），本应用聚焦于**真正掌握单词的使用能力**——包括搭配、语境和地道表达，而不仅仅是"认识"单词。

**目标用户**：英专生（英语专业学生）
**目标平台**：Android（PWA 安装到桌面），后续可扩展 iOS
**当前阶段**：Demo（个人测试用）

---

## 功能需求

### 1. 词书系统

**优先级**：P0（核心）

- 预置固定词书：初中、高中、专升本、CET-4、CET-6、TEM-4、TEM-8
- 每本词书包含词书名称、中文名、描述、词汇量
- 用户可查看每本词书的学习进度（已完成 / 总词汇量）
- Demo 阶段：CET-4 + TEM-4 各 30-50 词

### 2. 单词学习流程

**优先级**：P0

- 单词卡片展示：单词、音标（IPA）、词性、中文释义
- 发音功能：点击 🔊 按钮播放单词发音（Web Speech API）
- 例句轮播：每个单词提供 3-4 个不同语境的例句（学术/日常/文学/商务）
  - 每条例句显示英文原句 + 中文翻译
  - 例句可收藏（加入复习本）
- 学习操作按钮：
  - "认识" → 艾宾浩斯阶段推进（轻松正确）
  - "模糊" → 艾宾浩斯阶段保持（正确但犹豫）
  - "不认识" → 艾宾浩斯阶段回退
- 每日学习进度显示："今日已学 X/Y"

### 3. 复习本（核心创新）

**优先级**：P0

- 收藏的例句全部进入复习本
- 每日晚间复盘：
  - 显示中文句子 + 黄色标注目标单词
  - 隐藏英文原句
  - 用户根据中文回忆并构造英文句子
  - 支持同义词替换（想不起原词时）
- AI 批改反馈（DeepSeek API）：
  1. 目标单词检查：是否使用正确、拼写是否正确、同义词列表
  2. 语法错误分析：逐项指出错误 + 纠正 + 中文解释
  3. 搭配问题分析：不自然的词组搭配 + 地道替代表达
  4. 原句地道表达：标注原句中的习语、搭配亮点
  5. 综合评分（0-100）
- 复习结果可一键添加到积累本

### 4. 积累本

**优先级**：P1

- 保存地道表达、习语搭配、用法笔记
- 来源：
  - AI 批改建议（一键添加）
  - 用户手动添加
- 每条包含：表达内容、例句、中文释义、标签、来源标识
- 支持按时间/标签浏览

### 5. 自测中心 🆕

**优先级**：P1

#### 5.1 听音辨意
- 播放单词发音 → 从 4 个中文释义中选择正确答案
- 用户可切换：显示拼写 / 隐藏拼写（自定义难度）
- 即时反馈：正确绿色提示 / 错误红色提示 + 显示正确答案
- 测试结束显示：正确率、错词列表

#### 5.2 填空拼写
- 显示例句（目标词挖空为 `_______`）+ 中文翻译辅助
- 用户输入缺失单词 → 拼写比对
- 测试结束显示：正确率、错误对比（"你的答案" vs "正确答案"）

### 6. 复习节奏

| 频率 | 内容 | 方式 |
|------|------|------|
| 每日 | 复习本 + 积累本 | 中→英造句 + AI 批改 |
| 每周 | 本周所学单词 | 认识选择 + 拼写输入 |
| 每月 | 本月复习单词 | TTS 播客（每词 2-3 遍，间隔 3 秒） |

### 7. 艾宾浩斯遗忘曲线调度 🆕

**优先级**：P0

- 采用 8 阶段艾宾浩斯复习间隔：
  0→20min→1h→1d→2d→6d→14d→30d→已掌握
- 学习行为自动推进/保持/回退阶段
- 预估每个单词的记忆保留率

### 8. 学习统计与反馈 🆕

**优先级**：P1

- 遗忘曲线可视化（理论曲线 vs 用户实际表现）
- 每日学习柱状图（新学/复习/自测三色区分）
- 掌握率环形图（new / learning / reviewing / mastered）
- 打卡日历热力图（类似 GitHub 贡献图）
- 学习摘要卡片（累计时长、掌握单词数、平均正确率、连续打卡）

### 9. 设置与数据管理

**优先级**：P1

- DeepSeek API Key 管理
- 每日学习目标设置（默认 20 词/天）
- 自测出题数量设置（默认 20 题）
- 数据导出（JSON 下载）/ 导入
- 清除所有数据（确认后）

### 10. PWA 离线能力

**优先级**：P1

- 可安装到 Android 桌面（全屏 standalone 模式）
- 静态资源离线缓存（Workbox precaching）
- 学习数据本地存储（IndexedDB）
- 离线提示条（断网时显示）
- AI 功能断网时延迟处理（联网后自动重试）

## 非功能需求

### 设计
- **配色**：黑白灰为主色调（#1A1A1A / #4B5563 / #9CA3AF / #FFFFFF / #F9FAFB / #E5E7EB）
- **强调色**：黄色（#F59E0B）仅用于目标单词标注
- **风格**：极简、干净、留白多

### 性能
- 首屏加载 < 3 秒（PWA 缓存后 < 1 秒）
- AI 请求响应 < 5 秒（DeepSeek API）
- 数据库查询 < 50ms（IndexedDB）

### 数据
- 所有数据存储在本地（IndexedDB），无需后端服务器
- API Key 仅存储在本地，不经过第三方
- 数据量 < 5MB（远在浏览器 50MB+ 限制内）

## 用户故事示例

1. **作为英专生**，我希望能选择 TEM-4 词书开始背单词，每个单词看到不同场景的例句
2. **作为英专生**，我希望能收藏感兴趣的例句，晚上复习时练习造句
3. **作为英专生**，我希望 AI 能指出我造句中的搭配问题和语法错误，帮我真正学会使用单词
4. **作为英专生**，我希望能把地道表达保存到积累本，方便随时查看
5. **作为英专生**，我希望看到自己的学习统计，了解记忆保留情况
6. **作为英专生**，我希望在没有网的时候也能背单词和复习
